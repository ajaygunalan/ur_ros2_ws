## UR ROS2 Workspace

### Setup

Clone:
```bash
git clone git@github.com:ajaygunalan/ur_ros2_ws.git
cd ur_ros2_ws
```

Install vcstool:
```bash
pipx install vcstool
```

Import packages:
```bash
vcs import src < workspace.repos
```

Open in container:
```bash
code .
```

Click "Reopen in Container" when prompted
